# principles/ — 全プロジェクト横断の一般法則

各プロジェクトの `learn.md` から**昇格**してきた、普遍的な知見を置く。
howWhatWhy 内の最上位の知識層。

- **昇格の方法**: 毎週の棚卸しエージェントが全 `learn.md` を読み、昇格案を出す（[[rule]] の「棚卸し」を参照）。
- **判定基準**: 同じ原則を指す例がおおむね2〜3個で原則化を検討（目安）。1例でも明らかに普遍なら可。
- **粒度**: 1ファイル1原則（原則＋それを支える具体例）。
- **接続**: `[[リンク]]` で互いに繋ぐ。

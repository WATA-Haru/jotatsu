# Inbox — 摩擦ゼロの捕捉先

思いついたら時刻付きで1行放り込む。**ルールなし**（what/why/learn を考えない）。
後で project の note/learn へ振り分けるか、新 project を起こすか、principle にするか、**捨てる**。
**溜める場所ではなく通り抜ける場所。** 7日超の未処理は処理 or 削除（月曜のポートフォリオ点検が促す）。

捕捉のショートカット（Obsidian）:
- どこからでも: **QuickAdd** の Capture をホットキー化（フォーマット `- {{date:YYYY-MM-DD HH:mm}} ` を inbox.md に追記）
- カーソル位置に日付: コア「**テンプレート**」プラグインの「現在の日付を挿入」をホットキー化

行のフォーマット: `- YYYY-MM-DD HH:mm <走り書き>`（先頭の日付で滞留を検出する）

---

- 2026-06-29 10:30 （例）ベースのBPM練習、メトロノーム録音で客観視できそう → learn 候補
